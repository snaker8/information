Teacher's Guide

ClassIn 리포트 × 오르조 연동 가이드

오르조 학습 풀이 데이터를 ClassIn 리포트에 연동하여
학생별 AI 분석 보고서를 자동 생성하는 프로세스 안내

이런 게 가능해집니다

오르조에서 학생이 푼 문제 데이터(정오답, 풀이시간, 판서 이미지)를 ClassIn 리포트에 자동으로 가져와서, AI가 학습 진단 · 강점 분석 · 학부모 메시지까지 자동 생성합니다. 선생님은 오르조 공유 URL만 붙여넣으면 됩니다.

진행 프로세스

1
오르조
오르조에서 학생 학습 현황 페이지 열기

오르조 학생 학습 현황 페이지에서 해당 학생의 풀이 결과를 확인합니다. 상단에 점수 · 정답 수 · 풀이시간 · 복습시간이 요약되어 있고, 아래 테이블에서 문제별 쓴 답 · 정답 · 풀이시간 · 점수를 볼 수 있습니다.

오르조 학생 학습 현황 — 문제별 풀이 데이터 및 우측 상단 '페이지 공유' 버튼
2
오르조
공유 URL 복사하기

오르조 학습 현황 페이지 우측 상단의 🔗 페이지 공유 버튼을 클릭하여 공유 URL을 복사합니다.

https://class.orzo.kr/public-reports/...

이 URL에 학생의 전체 풀이 데이터(정오답, 시간, 판서 이미지)가 포함되어 있습니다.

3
ClassIn 리포트
분석 시작 — URL 붙여넣기 & 자동 캡처

ClassIn 리포트의 START ANALYSIS 화면에서:

  • 학생 이름(Student Name)을 입력합니다.
  • 클래스명 · 과정명을 입력합니다.
  • URL AUTO CAPTURE 영역에 복사한 오르조 URL을 붙여넣고 자동 캡처 버튼을 누릅니다.

자동 캡처가 완료되면 오르조 풀이 페이지의 판서 이미지가 자동 추출됩니다.

URL 붙여넣기 화면
캡처 완료 — 판서 이미지 자동 추출
4
AI 자동 생성
AI 분석 보고서 자동 생성

분석이 완료되면 AI가 오르조 풀이 데이터 + 판서 이미지를 기반으로 학생별 맞춤 보고서를 자동 생성합니다.

  • Learning Progress — 학습 진행 상황 및 이해도 종합 평가
  • Key Strengths — 잘한 문제의 근거(Evidence)와 함께 강점 분석
  • Professional Advice — 선생님 시점의 전문 학습 조언
  • Message for Parents — 학부모님께 보내는 메시지 + Home Coaching Tips
  • Board Analytics — 전체 판서 이미지 갤러리
자동 생성된 AI 분석 보고서 — Learning Progress, Key Strengths, 학부모 메시지
5
학부모 공유
학부모 페이지에서 판서 상세 확인

생성된 보고서를 학부모님께 공유하면, 보고서 하단의 Board Analytics 영역에서 판서 이미지를 하나하나 클릭하여 확대 확인할 수 있습니다.

학부모님이 직접 자녀의 풀이 과정을 문제별로 살펴볼 수 있어, 어떤 문제에서 어떻게 풀었는지 구체적으로 파악할 수 있습니다.
