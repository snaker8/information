1. 수업모델 개요

과사람 수학학원은 "체계적 개념 전달"과 "데이터 기반 개별 약점 보완"을 하나의 시간표 안에 결합한 하이브리드 수업모델을 운영합니다. 강의식 수업의 체계적 커리큘럼과, 과외식 수업의 1:1 맞춤 관리를 동시에 제공하여 최상위권 학생과 전국단위 자사고 진학을 목표로 하는 학생에게 최적화된 학습 환경을 만듭니다.

핵심 구조: A/B 2트랙 교차 운영으로, 1인의 메인강사와 1인의 서브강사만으로 개념수업 · 유형학습 · 특강 · 개별클리닉 4가지 프로그램을 효율적으로 순환

9. 주간 수업 구성

과정 기간 설계: 중등 과정은 3개월(12주), 고등 과정은 내용 밀도를 고려하여 4개월(16주)로 운영.
정규 과정은 기본 → 응용까지, 심화는 특강으로 분리 운영합니다.
• 중등 5과정 × 3개월 = 15개월

12. 평가 시스템

과사람의 평가 체계는 "매일 확인 → 누적 점검 → 과정 총괄 → 전체 위치 파악"의 4단계로 구성됩니다. 모든 평가 결과는 데이터로 축적되어 학생별 약점 맵과 학부모 리포트의 근거 자료가 됩니다.

4단계 평가 체계

단계평가명주기내용담당
1단계 복습테스트 매일 지난 차시 핵심 개념 + 대표 유형 5~10문항 (5:00~5:30, 수업 시작 전 30분) 서브강사 채점·피드백
2단계 일일테스트 매일 당일 유형학습 후 이해도 확인 (7:00~8:30 유형학습 시간 내 실시). 오답은 오답 클리어존에서 유사 문제로 즉시 재풀이 → 0% 오답 원칙 메인 or 서브강사
3단계 단원 누적 테스트 2~3주마다 진행 중인 과정 내 누적 범위 출제. 이전 단원을 반복 포함하여 장기 기억을 유도하고 결손 단원을 즉시 발견 메인강사 출제

14. 학습 데이터 관리 시스템

클래스인 수업에서 생성되는 판서·영상 데이터를 학부모용 리포트와 학생용 아카이브로 가공하여 전달하는 자체 개발 시스템입니다. "보이지 않는 공부를 데이터로 증명"하는 과사람만의 기술적 차별화 포인트입니다.

솔루션 ① ClassIn 리포트 — 학부모 데이터 기반 확신 제공

클래스인 수업 판서(88×2000px 저해상도 PNG)를 AI가 자동 분석·재구성하여, 학부모가 한눈에 이해할 수 있는 학습 리포트로 변환합니다.

기능설명효과
판서 시각화가독성 낮은 긴 PNG를 크고 선명한 형태로 자동 재구성학부모 열람률 대폭 향상
AI 성취도 분석판서 내용을 AI가 분석하여 학습 성취도를 자동 도출"오늘 이만큼 해냈구나" 확신
정기 리포트 발송주간/월간 리포트를 학부모에게 자동 전달막연한 불안 → 데이터 확신
BEFORE → AFTER: 가독성 제로의 판서 PNG 원본 → AI 분석 학습 리포트(판서 시각화 + 성취도 분석 + 피드백). 학부모 반응: "공부하고 있겠지" → "오늘 이 단원에서 이만큼 성장했구나"

솔루션 ② ClassIn 아카이브 — 학생 개별화 복습 DB

수업 영상과 판서 데이터를 학생 개별로 정리하여, 효율적인 복습과 영구적인 학습 자산 축적을 가능하게 합니다.

기능설명효과
듀얼 판서 비교선생님 판서와 학생 판서를 한 화면에서 실시간 비교이해 격차 즉시 파악
무음 자동 편집영상 무음 구간 자동 스킵 → 핵심만 남긴 복습 영상복습 시간 획기적 단축
영구 학습 DB3개월 자동 삭제 없이 영구 보존되는 개별화 학습 자산퇴원 방지(Lock-in) 효과
BEFORE → AFTER: 편집 없는 긴 영상 탐색 + 3개월 후 데이터 소멸 → 효율적 핵심 영상 + 판서 비교 + 영구 보존. 학생의 자기주도 복습 역량을 근본적으로 강화합니다.

15. 학부모 소통 체계

과사람의 학부모 소통은 "데이터로 말하고, 정기적으로 연결하고, 즉시 대응한다"는 3원칙으로 운영됩니다. 에듀테크 시스템(13·14번)에서 생성되는 학습 데이터가 소통의 핵심 콘텐츠가 됩니다.

16. 입학 · 배치 프로세스

과사람은 학년이 아닌 과정 기준으로 반을 편성하므로, 신규 학생의 정확한 실력 진단과 적정 과정 배치가 특히 중요합니다.

입학 프로세스 5단계

① 상담 신청 → ② 레벨테스트 → ③ 결과 분석 → ④ 과정 배치 → ⑤ 첫 주 적응
단계내용상세담당
① 학부모 상담 학생 현재 학년·학교 진도·선행 이력·목표(일반고/자사고) 파악. 과사람의 과정별 시스템과 3개년 로드맵을 안내하여 학부모 이해도 확보 원장
② 레벨테스트 매쓰홀릭 진단평가 활용. 현행 학년 범위 + 선행 범위까지 출제. 단원별·유형별 성취도를 히트맵으로 자동 분석 메인강사 출제 / 시스템 채점
③ 결과 분석·과정 결정 단원분석 히트맵 기반으로 적정 시작 과정을 결정. 예: 중2 학생이지만 중3-1까지 양호 → 공통수학1 배치. 반대로 중2인데 중1-2 결손 → 중1-2부터 시작 원장 + 메인강사 협의
④ 과정 배치 · 시간표 확정 해당 과정 반의 현재 진도 확인 → 합류 시점 결정. 진도 차이가 크면 첫 1~2주 개별클리닉으로 보완 후 합류 원장
⑤ 첫 주 적응 관리 첫 주는 서브강사가 집중 케어. 복습테스트·일일테스트 결과를 면밀히 모니터링하고, 1주 후 학부모에게 적응 상황 리포트 발송 서브강사 + 원장 확인
배치 기준 핵심 원칙: "학년이 아니라 실력이 기준입니다." 중1이 공통수학1에 앉을 수도, 중3이 중2-2부터 시작할 수도 있습니다. 레벨테스트 데이터가 유일한 판단 근거이며, 학부모에게 히트맵을 직접 보여주며 설명합니다.

17. 내신 대비 운영

과사람은 선행 중심 학원이지만, 시험 기간에는 내신 대비를 병행합니다. 정규 선행 진도를 완전히 멈추지 않으면서, 내신 범위를 집중 대비하는 하이브리드 방식입니다.

내신 대비 운영 타임라인

시기구분운영 방식활용 도구
시험 4주 전 사전 준비 내신 해당 학생 파악 + 학교별 시험 범위 확인 + 매쓰홀릭에서 학교별·교과서별 기출 자료 준비 매쓰홀릭 교재연동
시험 3주 전 내신반 편성 내신 대비가 필요한 학생들을 별도 내신 특강반으로 편성. 정규 시간표의 특강·클리닉 슬롯(8:30~10:00)을 내신 대비 시간으로 전환 클래스인 그룹 분리
시험 2주 전 집중 대비 내신 학생: 정규 개념수업(5:30~7:00)은 유지 + 유형학습·클리닉 시간을 내신 범위 집중 풀이로 전환. 비내신 학생: 정규 진도 정상 운영 매쓰홀릭 시험지 제작
시험 1주 전 최종 점검 학교별 기출 변형 모의고사 실시(매쓰홀릭 쌍둥이 문제). 오답 클리어존 집중 운영. 서술형 대비 풀이 점검 매쓰홀릭 + 클래스인
시험 종료 후 복귀 시험 결과 분석 → 학부모 리포트 발송. 정규 선행 진도로 즉시 복귀. 내신 결과 데이터는 매쓰홀릭에 축적 매쓰홀릭 분석보고서
운영 원칙: "선행을 멈추지 않으면서 내신도 잡는다." 정규 개념수업(5:30~7:00)은 선행 진도를 유지하고, 유형학습·클리닉 시간(7:00~10:00)을 내신 집중 시간으로 유연하게 전환합니다. 내신이 없는 학생은 정규 진도를 그대로 진행하므로 시간 낭비가 없습니다.

내신 대비 × 매쓰홀릭 연동

시험 전
교과서별 기출 → 학교 맞춤 시험지 자동 생성
단원분석 히트맵으로 약점 단원 집중 배정
쌍둥이 유사 문제로 변형 대비
시험 후
실제 내신 점수 입력 → 예측 점수 대비 분석
오답 유형 데이터 → 다음 시험 대비 자료 축적
학부모에게 분석 리포트 자동 발송

18. 강사 채용 · 육성 기준

과사람의 하이브리드 수업모델은 메인강사(강의력)와 서브강사(밀착 관리력)의 역할이 명확히 분리되어 있습니다. 따라서 채용 기준도 역할별로 다르게 설계합니다.

역할별 채용 기준

메인강사 — 강의력 중심
항목기준
핵심 역량체계적 개념 전달력, 자체 커리큘럼 운영 능력, 판서 구조화 역량
경력중등·고등 수학 강의 경력 3년 이상 또는 이에 준하는 실력 검증
교재 역량개념쎈·일품·마플시너지 등 시중 교재 숙지 + 과정별 커리큘럼 설계 가능
에듀테크클래스인 전자칠판 활용 능력 (입사 후 교육 가능하나, 디지털 수업 친화력 필수)
채용 절차이력서 → 시범강의(30분) → 원장 심층면접 → 2주 수업 시범 운영
서브강사 — 밀착 관리력 중심
항목기준
핵심 역량학생 개별 약점 파악력, 꼼꼼한 오답 관리, 인내심과 소통 능력
자격수학 관련 전공 또는 수학 과외·학원 보조 경험. 신입도 가능(육성형 채용)
관리 역량매쓰홀릭 학습관리 시스템 운영, 클리닉 자료 제작, 학부모 소통
에듀테크매쓰홀릭·위드아카 시스템 운영 능력 (입사 후 집중 교육)
채용 절차이력서 → 1:1 학생 지도 시연(20분) → 원장 면접 → 1주 현장 OJT

강사 온보딩 · 교육 프로그램

시기구분메인강사서브강사
입사 1주 OJT 과사람 수업모델·시간표 구조 이해, 클래스인 전자칠판 실습, 기존 메인강사 수업 참관 매쓰홀릭·위드아카 시스템 교육, 오답 클리어존 운영법, 기존 서브강사 클리닉 참관
입사 2~4주 시범 운영 실제 반 배정 후 수업 진행, 원장이 수업 모니터링 및 피드백. 판서 구조화·개념맵 노트 지도법 코칭 실제 학생 클리닉 담당, 일일 리포트 작성 실습, 학부모 소통 롤플레이
입사 1개월~ 정식 배치 담당 과정·반 확정. 교재 라인업(8번)에 따른 커리큘럼 운영 시작 담당 반·학생 확정. 매쓰홀릭 주간 클리닉 생성 + 학부모 정기 소통 시작

강사 평가 · 유지 시스템

평가 항목내용주기
학부모 만족도 조사 수업 만족도, 소통 만족도, 자녀 성적 변화 체감도를 학부모 설문으로 조사. 결과는 강사별로 정리하여 피드백 분기별 (연 4회)
학생 성적 데이터 추적 담당 반 학생들의 누적테스트·제로테스트 평균 점수 추이, 오답률 감소 추이를 매쓰홀릭 데이터로 객관 평가 과정 종료 시마다
수업 모니터링 원장이 클래스인 녹화 영상 또는 실시간 참관으로 수업 품질 점검. 판서 구조화, 학생 참여 유도, 시간 배분 확인 월 1회 이상
원장 1:1 피드백 만족도 조사 + 성적 데이터 + 수업 모니터링 결과를 종합하여 원장이 강사별 1:1 면담 진행. 강점 유지 + 개선점 코칭 분기별
대우 원칙
유능한 강사가 오래 머물 수 있도록 업계 경쟁력 있는 급여 + 커리큘럼 자율권을 보장합니다. 메인강사는 자신의 교수법과 교재 선택에 자율성을 가지되, 과사람의 시간표 구조와 평가 체계 안에서 운영합니다.
성장 경로
서브강사 → 메인강사 전환 경로를 열어두어, 서브로 입사한 강사가 역량을 키워 메인강사로 성장할 수 있습니다. 전환 기준: 1년 이상 근무 + 시범 강의 통과 + 원장 추천.

19. 졸업 후 연결 경로

과사람 졸업 → 3가지 경로

경로 ① 과사람 특목관 고등부 이관
대상: 일반고 진학 학생 대부분
내용: 과사람 특목관 고등부로 자연스럽게 이관. 중등부에서 쌓은 학습 데이터와 진단 기록이 그대로 연결되어 끊김 없는 관리 지속.
장점: 동일 시스템 내 연속성 확보
경로 ② 온라인 수업 연결
대상: 청운고·자사고·특목고 진학 학생
내용: 과사람 온라인 수업 프로그램으로 연결. 현재 청운고·자사고 재학 중인 졸업생이 온라인으로 과사람 수업을 이어가고 있으며, 과사람의 관리 시스템 안에서 고등 수학 학습을 지속.
장점: 전국 어디서든 학습 연속성 유지 (검증된 모델)
경로 ③ 외부 고등 학원 이동
대상: 개별 선택에 따라
내용: 과사람에서 고등 수학을 완성한 상태이므로, 어떤 고등 학원을 가더라도 인강만으로도 1~2등급 커리를 소화할 수 있는 실력이 갖춰져 있음.
장점: 고등부 학원 선택의 자유
브랜딩 메시지: "과사람을 거친 학생은 어디를 가든 수학에서 앞서간다." 일반고에서는 최상위권, 자사고에서는 온라인으로 연결, 어떤 환경에서든 고등 수학의 기초가 완성된 상태로 출발합니다.
